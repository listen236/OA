# 页面模板规范

## 列表页组件结构

```tsx
<PageContainer title="员工信息">
  <SearchPanel />
  <ActionToolbar />
  <DataTable />
</PageContainer>
```

## 查询区字段建议

通用查询字段：

- 关键词
- 编号
- 名称
- 所属组织 / 部门
- 状态
- 日期范围

## 工具栏按钮

通用按钮：

- 重置
- 字段设置
- 刷新
- 导出
- 导入
- 新建

审批页面额外按钮：

- 发起审批
- 批量提交
- 撤回

考勤统计页面额外按钮：

- 生成
- 重算
- 确认
- 锁定

## 行操作

一般：

- 查看
- 编辑
- 删除

复杂业务：

- 发起审批
- 确认
- 放弃
- 撤回
- 导出

超过三个操作时使用：

```tsx
<Dropdown menu={{ items }}>
  <Button type="link">更多</Button>
</Dropdown>
```

## 表单页结构

```tsx
<Drawer title="新增员工" width={960}>
  <Form layout="horizontal">
    <Card title="基础信息">
      <Row gutter={16}>
        <Col span={12}>...</Col>
        <Col span={12}>...</Col>
      </Row>
    </Card>
    <Card title="任职信息">...</Card>
    <Card title="附件信息">...</Card>
  </Form>
</Drawer>
```
