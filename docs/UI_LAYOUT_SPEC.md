# UI 布局规范

## 1. 整体结构

采用企业级 ERP 后台布局。

```text
+--------------------------------------------------------------------------------+
| Header：LOGO / 系统名                                      BI / 消息 / 用户     |
+--------------------------------------------------------------------------------+
| SideMenu | MultiTabs                                                           |
|          +---------------------------------------------------------------------+
|          | Page Content                                                        |
|          |                                                                     |
|          |                                                                     |
+----------+---------------------------------------------------------------------+
```

## 2. Header

高度：`56px`

内容：

```text
左侧：智慧工厂AIF
右侧：BI入口 / 消息通知 / 用户头像 / admin下拉
```

样式：

- 白底
- 底部浅灰边框
- 字号 18px 左右
- 顶部固定

## 4. 多标签页

位于 Header 下方、内容区顶部。

参考截图样式：

```text
+------+--------+--------+--------+----------------+
|首页  |员工信息|组织管理|考勤组  |薪资审批        |
+------+--------+--------+--------+----------------+
```

要求：

- 点击菜单打开新 Tab
- 重复点击已有页面时切换到该 Tab
- Tab 可关闭
- 首页不可关闭
- 使用 Ant Design Tabs 的 `editable-card` 风格

## 6. 表单页模板

参考截图表单页。

```text
基础信息       更多信息       联系人
------------------------------------------------
字段A              字段B
字段C              字段D
上传附件
字段E              字段F
字段G              字段H

底部固定按钮：重置 / 提交 / 批量提交
```

规则：

- 两列布局为主
- 重要必填字段左侧显示红色星号
- 上传区域使用虚线边框
- 底部按钮固定在页面底部右侧
